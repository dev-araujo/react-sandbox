## Como usar

### Clonar o repositório

```bash
git clone https://github.com/dev-araujo/react-sandbox.git
cd react-sandbox
```

### Executar um projeto

```bash
cd pokedex-rxjs
npm install
npm run dev
```

Acesse `http://localhost:5173` no navegador.
